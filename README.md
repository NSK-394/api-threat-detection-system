# API Abuse Detection (ML + Behavioral Analysis)

This project detects abusive API usage using behavior-based fingerprinting and machine learning anomaly detection. It was built during a cybersecurity hackathon.

Core components:
- API log simulator (normal + bot traffic)
- Per–API-key behavior fingerprinting
- Feature engineering pipeline
- Elliptic Envelop anomaly model
- Risk scoring decision engine
- Real-time dashboard for analysts

Tech stack: Python, Scikit-learn, Pandas, Flask ect
